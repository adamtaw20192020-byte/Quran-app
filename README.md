# تطبيق القرآن الكريم

تطبيق Flutter احترافي لعرض القرآن الكريم مع الترجمة، البحث، والمفضلة.
يعتمد على [alquran.cloud API](https://alquran.cloud/api) مجاناً وبدون مفتاح API.

## المميزات
- عرض جميع السور (114 سورة) مع تفاصيلها
- عرض آيات كل سورة بالنص العثماني + ترجمة إنجليزية (قابلة للتبديل)
- بحث نصي داخل القرآن كامل
- حفظ آيات كمفضلة محلياً على الجهاز (بدون إنترنت)
- دعم كامل للغة العربية واتجاه RTL
- دعم الوضع الليلي (Dark Mode) تلقائياً حسب إعدادات الجهاز
- معالجة كاملة للأخطاء (فقدان الإنترنت، فشل الخادم) مع إمكانية إعادة المحاولة

## هيكل المشروع
```
lib/
  main.dart                    نقطة البداية + الثيم العام
  models/
    surah.dart                 نموذج بيانات السورة
    ayah.dart                  نموذج بيانات الآية
  services/
    quran_api.dart             الاتصال بـ API
    bookmark_service.dart      حفظ المفضلة محلياً
  screens/
    home_screen.dart           قائمة السور
    surah_screen.dart          عرض آيات سورة
    search_screen.dart         البحث
    bookmarks_screen.dart      المفضلة
  widgets/
    ayah_card.dart             بطاقة عرض الآية
```

## طريقة التشغيل (بما إنك تعمل من الجوال)

بما إنك ما عندك كمبيوتر لتثبيت Flutter SDK، عندك 3 خيارات:

### الخيار 1: Replit (الأسهل من الجوال)
1. افتح replit.com من متصفح الجوال وسجل حساب
2. أنشئ Repl جديد واختر قالب Flutter
3. ارفع كل الملفات اللي أعطيتك إياها بنفس الهيكل
4. شغّل الأمر `flutter run -d web-server` من الـ Console داخل Replit

### الخيار 2: Codemagic أو GitHub Actions (لبناء APK فعلي)
1. ارفع المشروع كامل إلى مستودع GitHub (تقدر تسوي هذا من تطبيق GitHub على الجوال)
2. اربط المستودع بـ Codemagic.io (فيه خطة مجانية)
3. Codemagic يبني لك APK جاهز للتحميل والتثبيت مباشرة، كل هذا من السحابة بدون أي تثبيت محلي

### الخيار 3: تحويله لمشروع FlutterFlow
إذا حبيت واجهة سحب وإفلات بدل التعامل مع الكود مباشرة، أقدر أساعدك تحول نفس فكرة الشاشات (قائمة سور، شاشة سورة، بحث، مفضلة) لخطوات داخل FlutterFlow وتربطه بنفس الـ API.

## ملاحظة مهمة
الكود مكتوب ومُدقق منطقياً بالكامل، لكن ما قدرت أشغّله فعلياً (compile) لأن بيئتي الحالية ما فيها Flutter SDK مثبت. أنصحك تجرب البناء عبر Codemagic أول شي للتأكد، وإذا طلعت أي رسالة خطأ ارسلها لي وأصلحها فوراً.
